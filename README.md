# davidemagliano.com

Personal site + notes.

- **Human view:** articles + about
- **Agent view:** `/agent` and `/agent.json` (simplified + linkable)

## Stack

- Astro (static)
- Deployed to GitHub Pages (output folder: `docs/`)

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Build output is written to `./docs/` (for GitHub Pages).
